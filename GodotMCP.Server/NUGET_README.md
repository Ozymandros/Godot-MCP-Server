## Godot MCP Server

`GodotMCP.Server` is a .NET global tool that exposes Godot 4.x automation over MCP (Model Context Protocol) using stdio JSON-RPC.

### Install

```bash
dotnet tool install --global GodotMCP.Server
```

### Run

```bash
godot-mcp
```

### Prerequisites

- .NET 10 runtime
- Godot 4.x binary available via `GODOT_PATH` or `PATH`
