# VS Code Dev Container Setup

This repository includes `.devcontainer` support for consistent local development.

## Open in container

1. Install Dev Containers extension in VS Code.
2. Open the repository.
3. Run "Dev Containers: Reopen in Container".

## What it provides

- .NET 10 SDK environment
- Auto-restore on container creation
- C# extension preinstalled

## Notes

- Dev container is for development workflow, not production runtime.
- MCP stdio behavior remains the same inside or outside container.
