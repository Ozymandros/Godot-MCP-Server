# Godot MCP Server - Documentation

This folder contains developer guides, operation contracts, and testing guidance.

Files:

- `operations-runner.md` - JSON request/response contract used by the embedded GDScript runner.
- `testing.md` - How to run unit and gated integration tests locally and in CI.
- `CONTRIBUTING.md` - Contribution and code style guidelines.
- `release-checklist.md` - Steps and gates for releasing the tool and container images.
